书籍阅读平台

胡卓华 7月 08, 2018

这是怎样的一个平台？

  1. 收集所有不涉及版权的书籍供用户免费阅读，如“四大名著”、《资治通鉴》等。

  2. 收集有版权的书籍，将免费试读部分供用户阅读，收费部分付费阅读。

与应用商店上已有的阅读平台有何不同？

  1. 读书平台有很多种：

    1. 小说类网站有“起点网”等。
    2. 购书类网站有“卓越网”、“当当网”等。
    3. 综合类阅读网站有“腾讯阅读”、“爱奇艺阅读”等。
    4. 书评类网站有“豆瓣网”等。

  2. “腾讯阅读”书籍种类多，但收费类书籍只能用钱购买。“爱奇艺阅读”书籍种类少，但有“用户阅读时长”系统，用户可以通过阅读时长值来兑换任意书籍。

  3. 本平台属于综合类书籍平台，与“腾讯阅读”、“爱奇艺阅读”类似。结合二者所长，收集尽可能多的书籍，建立“用户阅读时长”系统，培养用户每天阅读习惯，鼓励用户多阅读，通过“阅读值”来兑换任意书籍。

  4. 鉴于绝大多数阅读软件网络小说类书籍占多数，而且与出版类书籍混合在一起，严重影响阅读质量。本平台严格区分网络小说类与出版类书籍，让用户更方便地找到好书并专注阅读好书，提高阅读质量。

既然与现有阅读平台相似，为什么还要打造这样的平台？

  1. 现有平台的数据库是属于这些公司的，而这些公司还没有做到数据开放。

  2. 现有的阅读平台还只停留在网络小说的创作和消费上。

  3. 数据即资源，当本平台拥有大量资源后，将对免费资源进行数据开放，利用大数据分析技术，可进一步进行学术研究。

  4. 本平台不局限于网络小说的创作，将收纳大量学术文献，衍生出更多的知识平台，如论文写作平台等。

平台发展方向

  1. 目标收集全球所有书籍，搭建阅读平台。

  2. 根据发展需要，衍生以下平台：

    1. 搭建实体书籍共享平台，实现同城共享阅读。
    2. 开发语音功能，帮助有需要的人通过“听”来获取书籍内容。
    3. 搭建创作平台，创作网络小说、毕业学术论文等，供用户发布创作并分成流量红利。
    4. 搭建大数据分析平台，基于大量书籍进行分析，获取靠人工难以分析出的信息。
    5. 借鉴kindle，研发属于中国制造的电子书阅读平板电脑。
    6. 筹建各地区阅读空间，为各城市营造特色阅读场所。
    7. 筹建图书出版社，将平台上的资源整合出版。
    8. 筹建慈善公益平台。
  3. 致力打造书友圈，促进爱阅读的人线上线下交流，分享阅读心得，推荐优秀书籍。

从现在开始，为这一愿景奋斗
